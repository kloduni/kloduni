### 🚀 What I'm Working On
#### [MedShop 2.0](https://github.com/kloduni/MedShop)
🧩 A hands-on e-commerce project I am building to strengthen my understanding of ASP.NET Core MVC and modern design patterns.  
**What I'm implementing:**
- **N-Tier Architecture:** Separating business logic, data access, and UI for cleaner code.
- **Security Mechanics:** Experimenting with Role-Based Access Control (RBAC) and CSRF protection.
- **Modernizing the UI:** Transitioning from Bootstrap to a custom Tailwind CSS theme, and using Vanilla JS to make cart interactions feel seamless without page reloads.
